Instruksi Instalasi HEN PRO CM — Ringkas

Halaman ini responsif (bisa dibuka di PC atau HP). Setiap langkah disertai placeholder screenshot 16:9 — ganti dengan screenshot nyata Anda jika perlu.

1. Download HFW dan masukan HFW ke Flashdisk/PS3/UPDATE/(disini)

Link: Google Drive — HFW

Letakkan file HFW di folder PS3/UPDATE/ pada flashdisk (FAT32-MBR).

2. Dan juga file HEN.p3t ke Flashdisk/4.92/(disini)

Letakkan HEN.p3t ke folder 4.92/ di root flashdisk sehingga struktur jadi 4.92/HEN.p3t.

3. Instal HFW seperti biasa

Ikuti proses instalasi HFW pada PS3 Anda sesuai petunjuk HFW. Pastikan PS3 dalam kondisi normal (BT/BD tidak rusak).

4. Konek ke internet

Sambungkan PS3 ke jaringan internet sebelum melanjutkan.

5. Setting tanggal dan waktu via internet (zone Bangkok)

Atur region/timezone ke Bangkok (UTC+7) dan aktifkan sinkronisasi waktu via internet.

6. Akses Web yg saya berikan (xstorm.xo.je)

URL: http://xstorm.xo.je

Buka alamat tersebut di browser PS3 dan ikuti instruksi di halaman. Jika web tertutup, hubungi penyedia untuk membuka kembali.

7. Tunggu Instalasi sampai selesai dan PS3 akan reboot otomatis

Setelah proses selesai PS3 biasanya reboot otomatis. Jika tidak, lakukan restart manual.

Note

  • PS3 wajib normal tanpa kerusakan BT maupun BD.
  • Flashdisk wajib FAT32-MBR.
  • Di dalam flashdisk wajib ada folder PS3 dan 4.92.
  • Jika web tertutup — hubungi penyedia untuk membuka kembali.
